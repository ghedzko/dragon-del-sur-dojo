# Técnicas Básicas de Karate para Principiantes

El karate es un arte marcial que requiere dedicación, disciplina y una sólida comprensión de las técnicas fundamentales. En este artículo, exploraremos las bases esenciales que todo principiante debe conocer.

## 1. Posiciones (Dachi-Waza)

Las posiciones son la base de todas las técnicas de karate. Algunas posiciones fundamentales incluyen:

- **Zenkutsu-dachi** (posición frontal)
- **Kokutsu-dachi** (posición hacia atrás)
- **Kiba-dachi** (posición de jinete)

![Posiciones básicas de karate](/images/karate-stances.jpg)

## 2. Técnicas de Puño (Tsuki-Waza)

El golpe de puño es una de las técnicas más básicas pero importantes:

- Choku-zuki (golpe directo)
- Gyaku-zuki (golpe inverso)
- Kizami-zuki (golpe adelantado)

## 3. Consejos para la Práctica

1. Mantén una respiración constante
2. Concéntrate en la forma antes que en la velocidad
3. Practica regularmente frente a un espejo
4. No te desanimes si no sale perfecto al principio

> "El karate es como agua hirviendo; si no le das calor continuamente, se vuelve fría." - Gichin Funakoshi
